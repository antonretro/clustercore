# 🔳 CLUSTER CORE

**Cluster Core** is a modern arcade puzzle experience built for high-speed tactical play and visual satisfaction. 

## 🕹️ Gameplay Mechanics
- **4+ Cluster Matching**: Connect four or more blocks of the same color in any shape to clear them.
- **Directional Arrows**: Metal pieces act as strategic anchors—they only clear if part of a horizontal or vertical line of 4+ in their specific direction.
- **Power-Ups**: Utilize **Bombs** for area-of-effect destruction and **Drills** to clear entire columns.
- **Fever Mode**: Build up your jackpot meter to trigger a high-scoring frenzy with double points and warp speed visual effects.

## ✨ High-Fidelity "Juice"
- **Dynamic Squash & Stretch**: Tactile piece movement and impact scaling.
- **Retro CRT Aesthetics**: Integrated scanline effects and vibrant neon color palettes.
- **Wavy UI Text**: High-energy feedback for "MEGA CLEARS" and combos.
- **Particle Systems**: Explosive color-matched feedback for every block cleared.

## 🛠️ Technical Details
- **Engine**: GameMaker
- **Architecture**: Modular script-based logic (`scr_grid_physics`, `scr_match_engine`, `scr_juice_systems`).
- **Resolution**: Scaled for crisp 240p-pixel-art integrity (4x scale).
